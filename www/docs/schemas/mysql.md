id: mysql
title: MySQL
---

The schema generated for a MySQL database when using the built-in models.

:::note
When using a MySQL database with the default adapter the timezone is set to `Z` (aka Zulu Time / UTC) and all timestamps are stored in UTC.
:::

## User

```json
"users": {
  "id": {
    "type": "int",
    "nullable": false
  },
  "name": {
    "type": "varchar(255)",
    "nullable": true,
    "default": null
  },
  "email": {
    "type": "varchar(255)",
    "nullable": true,
    "default": null
  },
  "email_verified": {
    "type": "timestamp",
    "nullable": true,
    "default": null
  },
  "image": {
    "type": "varchar(255)",
    "nullable": true,
    "default": null
  },
  "created_at": {
    "type": "timestamp(6)",
    "nullable": false
  },
  "updated_at": {
    "type": "timestamp(6)",
    "nullable": false
  }
}
```
